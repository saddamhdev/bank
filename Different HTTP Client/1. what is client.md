# 🧠 What Is a Client?

A **client** is simply:

> A system (or program) that sends a request to another system (called a server) to get something.

---

# 🏗 Simple Example

When you open:

* Chrome → visit `google.com`

Here:

* **Chrome = Client**
* **Google Server = Server**

Client → sends request
Server → sends response

---

# 📡 In Backend / Microservices

If `order-service` calls `user-service`:

```text
order-service  →  user-service
   (client)         (server)
```

Even though both are servers,
when one calls another → the caller becomes the **client**.

---

# 🔥 In Spring Boot Context

When you use:

* RestClient
* WebClient
* Feign
* gRPC

They are all **HTTP / RPC clients**

Because they send requests to other services.

Example:

```java
User user = userClient.getUser(1L);
```

Here:

* `userClient` = client
* `user-service` = server

---

# 🎯 Simple Real-Life Analogy

Restaurant example:

* You (customer) → Client
* Restaurant → Server
* Order food → Request
* Food delivered → Response

# 🏢 In Microservice Architecture

Every service can be:

* Server (when receiving request)
* Client (when sending request)

So role depends on situation.
